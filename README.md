## features

* [x] Local DynamoDB (port 8000)
* [ ] Built-in Local DynamoDB (can't spawn after build)
* View
  * Table view
    * [x] Sub collection view
    * [x] Table schema view
    * [ ] Nested JSON(Enter, Leave)
  * [x] JSON view
* Operation
  * Table
    * [ ] Add Table
    * [ ] Edit Table
    * [ ] Delete Table
  * Record
    * [x] Add record
    * [x] Edit record
    * [ ] Delete record
